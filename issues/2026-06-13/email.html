#01 · Tier 1 旗舰正式论文 · Science Robotics · 2026-06-10
Precise aggressive aerial maneuvers with sensorimotor policies
针对轻量无人机的极限机动受限于传感负载、反应时延和 SE(3) 姿态约束，论文通过模型规划轨迹初始化受限解空间，再通过强化学习和端到端策略蒸馏，并验证Science Robotics。
#02 · Tier 1 旗舰正式论文 · Nature Machine Intelligence · 2026-05-12
Peripheral control enabled by distributed sensing in an octopus-inspired soft robotic arm for autonomous underwater grasping
针对软体水下抓取面对形变、接触不确定和流体扰动，论文通过每个人工吸盘嵌入 LED 和 phototransistor，通过反射光估计接触力与方向，并验证Nature Machine Intelligence。
#03 · Tier 1 旗舰正式论文 · Science Robotics · 2026-04-22
Object-centric task representation and transfer using diffused orientation fields
针对曲面没有统一全局坐标系，论文通过方法用由偏微分方程控制的扩散过程，从原始点云和关键点在线计算平滑局部参考框架，再用这些 orientation fields 表达和迁移物体中心任务，并验证Science Robotics；摘要报告在几何、拓扑和关键点扰动下评估，并展示覆盖、切割、剥离等连续物理交互任务迁移。
#04 · Tier 2 高质量正式论文 · Nature Communications · 2024-09-27
Birdlike broadband neuromorphic visual sensor arrays for fusion imaging
针对可穿戴或嵌入式视觉系统受限于功耗、柔性集成和后端计算搬运，论文通过构建 P3HT/GaAs 纳米线 P-N 结并结合 Schottky 结，并验证Nature Communications。
#05 · Tier 2 高质量正式论文 · Nature Communications · 2024-08-07
Interspecies-chimera machine vision with polarimetry for real-time navigation and anti-glare pattern recognition
针对现有机器视觉多模仿人眼强度成像，缺少偏振信息；而偏振对抗眩光、天光导航和真实图像恢复都很有价值，论文通过ReS2/GeSe2 范德华异质结构成 optically controlled polarimetry memtransistor，并验证Nature Communications，摘要报告该器件可识别天体偏振用于实时导航。
#06 · Tier 2 高质量正式论文 · Nature Communications · 2024-01-29
Adaptive tactile interaction transfer via digitally embroidered smart gloves
针对触觉信息难以跨时间和空间传递，因为高覆盖传感、可穿戴舒适性、触觉反馈和个体差异校准通常无法同时满足，论文通过数字刺绣把压阻力传感器和振动触觉执行器嵌入纺织手套，再用机器学习建模个体对触觉反馈的反应并优化反馈参数，并验证Nature Communications，摘要报告手套可记录、再现和转移触觉交互。
#07 · Tier 2 高质量正式论文 · Nature Communications · 2023-09-27
Embedded shape morphing for morphologically adaptive robots
针对形态可变机器人想适应多任务环境，但传统方案常依赖外部气源、线缆或笨重支撑，无法真正成为可自主部署的身体结构，论文通过提出嵌入式 shape-morphing scheme，把形状驱动、形状感知和锁定机制整合进模块，并在自感知抓手、可变形四足身体和两栖无绳机器人中展示，并验证Nature Communications，摘要展示自感知变形抓手、可切换行走/爬行/水平攀爬的四足机器人。
#08 · arXiv / preprint · arXiv · 2026-06-11
Bounding Boxes as Goals: Language-Conditioned Grasping via Neuro-Symbolic Planning
针对开放词表桌面抓取的瓶颈不是只理解句子，论文通过框架用预训练 VLM 解析自然语言，生成 neuro-symbolic goal states，并验证90 次真实机器人试验、三档难度下总体成功率 73.3%，且无需任务特定训练。
#09 · arXiv / preprint · arXiv · 2026-06-11
SPARC: Reliable Spatial Annotations from Robot Demonstrations at Scale
针对具身基础模型需要大量空间标注，论文通过SPARC 利用机器人任务的时空结构生成 bounding boxes、物体轨迹和 manipulation phase labels，并验证摘要称在 1.7k 人工标注示范上，相比 detection-only baseline 提高定位精度。
#10 · arXiv / preprint · arXiv · 2026-06-11
See Selectively, Act Adaptively: Dual-Level Structural Decomposition for Bimanual Robot Manipulation
针对双臂操作中，论文通过View-Selective Visual Router 动态调节腕部视角贡献，并验证在 RoboTwin 2.0 六个仿真任务和三个真实长时程任务中。
